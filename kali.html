Kali Linux

OS for Hackers and pentesters

Kali Linux is a debian linux based distribution designed for info_security professionals. It offers an easier and smoother experience with vulnerablity findings and security examination. Kali contain hundreds of tools for various security tasks such as penetration testing, reverse engineering and social engineering. Kali Linux is developed, funded and maintained by Offensive Security.

Virtual machine

Requirements

  1. Have a recent version of vmware or virtual box. (Virtual box, Vmware)
  2. Have at least 8Gb of free storage in hard-drive.
  3. Download the suitable kali image for your virtual box here.

Installation Process [Vmware]

  1. Extract and open the image folder you just downloaded.
  2. Click on the .vmx file.
  3. Choose "I copied it" if vmware ask weither if you copied or moved it.
  4. Default username: root Default password: toor

Installation Process [Virtualbox]

  1. Open the .ova file you just downloaded.
  2. Click on import.
  3. Start up Kali linux
  4. Default username: root Default password: toor
